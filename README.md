# postxml-ng-valid
[![npm version][npm-image]][npm-url]

> [PostXML] plugin that replace `ng-` with `data-ng`.

## Installation
`npm i postxml-ng-valid --save-dev`

## Usage
[Postxml]

## Example

### Input
```html
<div ng-bind="data"></div>
```

### Output
```html
<div data-ng-bind="data"></div>
```
